# Play-Board Feature

Dieses Feature kapselt alle Komponenten und Services für den Spielmodus des Bingo-Boards.

- Komponenten: `play-board.component.ts`, `playable-board.component.ts`
- State: `playable-bingo-state.service.ts`
- Nutzung: Importiere aus `features/play-board` für alles, was den Spielmodus betrifft.
