# Edit-Board Feature

Dieses Feature kapselt alle Komponenten und Services für den Editiermodus des Bingo-Boards.

- Komponenten: `edit-board.component.ts`, `editable-board.component.ts`
- State: `editable-bingo-state.service.ts`
- Nutzung: Importiere aus `features/edit-board` für alles, was den Editiermodus betrifft.
